# 🖥️ Build Your First Web App 
This is a starter template to help you build your first web application using:

- Python (Flask)
- HTML
- CSS
- JavaScript

You will use this template to create your own project and begin building your tech portfolio 🚀

---

## ✨ Features
- Modern landing page (Bootstrap + Google Fonts)
- Navbar + hero banner
- Input ➜ Output forms
- API playground (random advice + random dog image)

---

## 🚀 Create your projects repository
Use this repository as a template 
1. Click **"Use this template"**  
2. Select **"Create a new repository"**  
3. Rename it to your project name  

Examples:
- `my-portfolio-app`
- `study-planner`

Your repository URL will look like:  
👉 https://github.com/Your-Username/Project-Name>  

Example:  
👉 https://github.com/michelletemi/Workshop-Flask-Template  


---

## 💻 Run locally (Visual Studio Code)

### 1. Clone the repo:
- Open Visual Studio Code
- Open Terminal (Terminal > New Terminal)
- Run:

```bash
git clone https://github.com/<your-username>/<projectname>
cd <project-name>
```

### 2. Run the application
In the terminal, run:

```bash
python main.py
```

- A URL will appear in the terminal (e.g. http://127.0.0.1:5000)  
- Copy and paste it into your browser  
- You should now see your app running 🎉  

---

## 🌐 Alternative: Run on Replit

If you don’t want to install anything locally, you can run this in your browser:

1. Go to 👉 https://replit.com  
2. Click **"Import code or design"**  
3. Select **GitHub**  
4. Paste your repository link  
5. Click **Import**  
6. Click **Run**  
7. Open the public URL to view your app
