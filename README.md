# 🖥️ Build Your First Web App 
This is a starter template to help you build your first web application using:

- Python (Flask)
- HTML
- CSS
- JavaScript

You will use this template to create your own project and begin building your tech portfolio 🚀

---

## ✨ Features
- Modern landing page (Bootstrap + Google Fonts)
- Navbar + hero banner
- Input ➜ Output forms
- API playground (random advice + random dog image)

---

## 🚀 Create your projects repository
Use this repository as a template 
    - Click **'Use this template'** 
    - Select **'Create a new repository'** 
    - Rename it to your project name 
    Example
        - my-portfolio-app
        - study-planner

Your repository URL will be: https://github.com/<your-username>/<projectname>
For example: https://github.com/michelletemi/Workshop-Flask-Template

---

## 🌐 Alternative: Run on Replit

1. If you don’t want to install anything locally:
2. 1. Go to [Replit](https://replit.com).
